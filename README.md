## 组件

> 3步骤：
>
> > * 创建组件构造器 
> >
> >   > 调用 **`Vue.extend()`** 方法，创建组件构造器
> >
> > * 注册组件
> >
> >   > 调用 **`Vue.component()`** 方法，注册组件
> >
> > * 使用组件
> >
> >   > 在Vue实例的作用范围内，使用组件
>
> 全局组件、局部组件
>
> 组件模板的分离
>
> 组件可以直接访问Vue实例中的数据嘛？
>
> > 组件，是一个单独功能模块的封装：即这个模块有自己的HTML模板( **`template`** 属性)、以及数据data( **`data()`** )。
> >
> > 组件里的 **`data()`** 必须返回一个对象。为什么必须是 **`data()`** 方法，而不是 **`data: {}`** ？
